## Hello World - input()

```
message = 'hello'

name = input('What is your name? ')

print(f'{message} {name}')
```

## While True Loop

```
message = 'hello'

while True:
    name = input('What is your name? ')
    print(f'{message} {name}')
```

## Change Data Types

```
while True:
    num1 = float(input('Number 1: '))
    num2 = float(input('Number 2: '))
    total = num1 + num2
    print(total)

```
